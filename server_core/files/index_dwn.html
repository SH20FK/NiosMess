Архитектура приватности

Два режима. Один понятный выбор.

Для повседневной переписки — синхронизация между устройствами. Для чувствительных разговоров — отдельный секретный чат со сквозным шифрованием.

Обычный чат
Файлы шифруются и передаются через защищённые облачные серверы NiosMess, чтобы их можно было быстро открыть на любом вашем устройстве.
Секретный чат
Передача происходит через облачные сервера NiosMess, шифруются по протоколу E2E что защищает вашу переписку, а криптографические ключи хранятся исключительно на устройствах участников.

Поддержка

Коротко о важном.

Ответы на вопросы о хранении файлов и работе NiosMess при нестабильном соединении.

В обычных чатах файлы шифруются и передаются через защищённые облачные серверы NiosMess для быстрой загрузки на любом вашем устройстве. В секретных чатах передача происходит так-же через сервер, но сообщения шифруются по протоколу Е2Е и даже NiosMess не может их прочитать.

Да. Архитектура передачи сетевых пакетов оптимизирована для условий с высокой потерей пакетов, что помогает сохранять доставку сообщений и стабильный звук даже при слабом Edge/3G-соединении.
