# NiosBot — Python библиотека для ботов

Простая библиотека, похожая на `telebot`, для создания ботов в NiosMess.

## Установка

Положите папку `niosbot` рядом с вашим скриптом (или в PYTHONPATH). Зависимость: `requests`.

```python
from niosbot import Bot, InlineKeyboardMarkup, InlineKeyboardButton

bot = Bot("YOUR_BOT_TOKEN")

@bot.message_handler()
def on_message(msg):
    bot.send_message(msg.chat_id, f"Привет, {msg.content}!")

bot.infinity_polling()
```

## API методы

- `bot.send_message(chat_id, text, reply_markup=None)` — отправить сообщение
- `bot.edit_message_reply_markup(chat_id, message_id, reply_markup)` — изменить клавиатуру
- `bot.delete_message(chat_id, message_id)` — удалить сообщение
- `bot.answer_callback_query(callback_query_id, text=None)` — ответить на callback query
- `bot.get_updates(offset=0, limit=100)` — получить updates (polling)

## Inline Keyboard

```python
from niosbot import InlineKeyboardMarkup, InlineKeyboardButton

markup = InlineKeyboardMarkup()
markup.add(
    InlineKeyboardButton("Кнопка 1", callback_data="btn1"),
    InlineKeyboardButton("Кнопка 2", callback_data="btn2")
)

bot.send_message(chat_id, "Выберите:", reply_markup=markup)
```

## Обработчики

```python
@bot.message_handler()
def on_msg(msg):
    print(msg.content, msg.chat_id, msg.sender_id)

@bot.callback_query_handler()
def on_cb(cb):
    print(cb.data, cb.from_user)
    bot.answer_callback_query(cb.id)
```
