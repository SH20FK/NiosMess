# Предложения по API (по мотивам public-apis)

Ниже — список публичных API, которые можно разумно встроить в NiosMess. Для каждого указана идея применения. Все позиции взяты из списка public-apis.

## Текст, язык, модерация
- LibreTranslate — перевод сообщений/каналов и быстрый перевод цитат по запросу.
- Detect Language — автоопределение языка сообщений (для автоперевода и фильтров).
- Tisane — детект токсичности/абьюза в чатах (мягкая модерация и авто-флаги).
- Sentiment Analysis (MeaningCloud) — тональность сообщений для аналитики и анти-фрод сигналов.
- Hirak OCR — извлечение текста из изображений/сканов (поиск по картинкам).

## Верификация и анти-спам
- AbstractAPI Phone Validation / numverify — проверка номеров при регистрации/восстановлении.
- Email Validation (AbstractAPI) — проверка валидности почты при регистрации.
- Disify — отсев временных/одноразовых email-адресов.

## Безопасность устройства и окружения
- Application Environment Verification — проверка рута/подмены окружения на Android (risk-сигнал).

## Ссылки и гео
- CleanURI или Cutt.ly — короткие ссылки для инвайтов/шеринга.
- IP Geolocation (AbstractAPI) или ipstack — грубая геолокация для анти-фрода/логов безопасности.

## Файлы и медиа
- Web3 Storage — внешнее хранение больших вложений/архивов каналов.
- File.io — одноразовые ссылки для временных файлов.
- Imgbb — быстрый хостинг превью изображений.

## Заметки по внедрению
- У большинства API есть бесплатные лимиты, но часто требуется ключ.
- Лучше начать с 1-2 интеграций (языки + валидация), затем наращивать.
