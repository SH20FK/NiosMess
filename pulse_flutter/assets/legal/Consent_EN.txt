1. Subject of Consent
The User consents to the processing of the following personal data:
- Email address;
- Username and Display Name;
- Profile description ("About Me" / Bio);
- Profile image (Avatar);
- Network identifiers: IP address, device type and model, operating system, cookie files, and local storage data (LocalStorage, IndexedDB);

2. Purposes of Data Processing
Personal data is processed for the following purposes:
- Creating and managing the User's account in the Service;

- Moderating public content in compliance with the Terms of Service.

3. Storage and Protection of Data
- All personal data is hosted on secure servers located in the Netherlands (European Union) in accordance with the EU General Data Protection Regulation (GDPR).
